BASS.net_timeout¶

BASS.net_timeout¶

The time in milliseconds to wait for a server to respond to a connection request.

The default timeout is 5 seconds (5000 milliseconds). When playlist processing is enabled via the Bass4Py.bass.BASS.net_playlist option, the timeout applies to each playlist entry rather than the playlist as a whole.
