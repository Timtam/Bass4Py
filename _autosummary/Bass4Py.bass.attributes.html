Description

This package contains the main attribute types which can be found on i.e. Bass4Py.bass.ChannelBase, Bass4Py.bass.Channel, Bass4Py.bass.Music and Bass4Py.bass.Stream instances. You will most likely not interact with those classes directly, but use them when instanciated on one of those objects.

The following attributes are available:

AttributeBase

BoolAttribute

BytesAttribute

FloatAttribute

FloatListAttribute
