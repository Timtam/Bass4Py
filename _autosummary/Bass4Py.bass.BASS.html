BASS¶

class Bass4Py.bass.BASS¶

This class represents the main entrypoint to all Bass4Py related functionalities. It allows to control various global settings and gain access to output devices and input devices to playback audio data or record from various input sources. All global properties are also accessible through this class.

See input_devices or output_devices to get further access to the really important features.
