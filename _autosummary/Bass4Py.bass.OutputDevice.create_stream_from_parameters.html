OutputDevice.create_stream_from_parameters¶

OutputDevice.create_stream_from_parameters(self, DWORD freq, DWORD chans, DWORD flags=0, callback=None)¶

Creates a user sample stream.

Parameters

  • freq (int) – The default sample rate. The sample rate can be changed using Bass4Py.bass.Stream.frequency.

  • chans (int) – The number of channels… 1 = mono, 2 = stereo, 4 = quadraphonic, 6 = 5.1, 8 = 7.1.

  • flags (Bass4Py.constants.STREAM) – a combination of the Bass4Py.constants.STREAM flags

  • callback (callable) – the callback parameter can be either None, in which case the data for the stream needs to be pushed manually via Bass4Py.bass.Stream.put_data(), or it can be a callable that accepts two parameters, the corresponding Bass4Py.bass.Stream object and an integer length of data expected, and needs to return a bytes object that should contain as much data as the length parameter indicates to prevent buffer underflows.

Returns

stream

Return type

Bass4Py.bass.Stream

Raises

  • Bass4Py.exceptions.BassInitError – init() has not been successfully called.

  • Bass4Py.exceptions.BassNotAvailableError – The Bass4Py.constants.STREAM.AUTOFREE flag cannot be combined with the Bass4Py.constants.STREAM.DECODE flag.

  • Bass4Py.exceptions.BassFormatError – The sample format is not supported by the device/drivers. If the stream is more than stereo or the Bass4Py.constants.STREAM.FLOAT flag is used, it could be that they are not supported.

  • Bass4Py.exceptions.BassSpeakerError – The specified speaker flags are invalid. The device/drivers do not support them, they are attempting to assign a stereo stream to a mono speaker or 3D functionality is enabled.

  • Bass4Py.exceptions.BassMemoryError – There is insufficient memory.

  • Bass4Py.exceptions.BassNo3DError – Could not initialize 3D support.

  • Bass4Py.exceptions.BassUnknownError – Some other mystery problem!

Sample streams allow any sample data to be played through BASS, and are particularly useful for playing a large amount of sample data without requiring a large amount of memory. If you wish to play a sample format that BASS does not support, then you can create a stream and decode the sample data into it.
