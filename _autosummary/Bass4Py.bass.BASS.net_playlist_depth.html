BASS.net_playlist_depth¶

BASS.net_playlist_depth¶

Maximum nested playlist processing depth. 0 = do not process nested playlists.

When playlist processing is enabled via the Bass4Py.bass.BASS.net_playlist option, this option limits how deep into nested playlists Bass4Py.bass.OutputDevice.create_stream_from_url() will go. The default depth is 1, which means playlists within the root playlist will be processed, but not playlists within those playlists.

int
