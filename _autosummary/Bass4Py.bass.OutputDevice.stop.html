OutputDevice.stop¶

OutputDevice.stop(self)¶

Stops the output, stopping all musics/ samples/ streams on it.

Returns

success

Return type

bool

Raises

Bass4Py.exceptions.BassInitError – init() has not been successfully called.

This function can be used after pause() to stop the paused channels, so that they will not be resumed the next time start() is called.
