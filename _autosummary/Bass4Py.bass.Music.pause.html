Music.pause¶

Music.pause(self)¶

Pauses this channel.

Returns

success

Return type

bool

Raises

  • Bass4Py.exceptions.BassNoPlayError – The channel is not playing.

  • Bass4Py.exceptions.BassDecodeError – The channel is not playable; it is a “decoding channel”.

  • Bass4Py.exceptions.BassAlreadyError – The channel is already paused.

Use Bass4Py.bass.Channel.play() to resume a paused channel. stop() can be used to stop a paused channel.
