BASS.update¶

BASS.update(self, DWORD length)¶

Updates the Bass4Py.bass.Stream and Bass4Py.bass.Music channel playback buffers.

Parameters

length (int) – The amount of data to render, in milliseconds.

Returns

success

Return type

bool

Raises

Bass4Py.exceptions.BassNotAvailableError – Updating is already in progress.

When automatic updating is disabled, this function, Bass4Py.bass.Music.update() or Bass4Py.bass.Stream.update() needs to be used to keep the playback buffers updated. The length parameter should include some safety margin, in case the next update cycle gets delayed. For example, if calling this function every 100ms, 200 would be a reasonable length parameter.
