ChannelBase.get_position¶

ChannelBase.get_position(self, DWORD mode=_BASS_POS_BYTE, bool decode=False)¶

Retrieves the playback position of this channel.

Returns

the position as a tuple when mode is set to Bass4Py.constants.POSITION.MUSIC_ORDER, otherwise int

Return type

int or tuple

Parameters

  • mode (Bass4Py.constants.POSITION) –

    how to retrieve the position. One of the following:

    • Bass4Py.constants.POSITION.BYTE (default): Get the position in bytes.

    • Bass4Py.constants.POSITION.MUSIC_ORDER: get the position as order, row tuple. (available only for Bass4Py.bass.Music only)

  • decode (bool) – Get the decoding/rendering position, which may be ahead of the playback position due to buffering. This flag is unnecessary with decoding channels because the decoding position will always be given for them anyway, as they do not have playback buffers.

Raises

  • Bass4Py.exceptions.BassNotAvailableError – The requested position is not available.

  • Bass4Py.exceptions.BassUnknownError – Some other mystery problem!
