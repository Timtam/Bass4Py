BASS.input_devices¶

BASS.input_devices¶

A InputDeviceEnumerator instance, allowing you to access all the different input devices available on your system. You can access its default attribute to quickly get access to the current default device on your system, or use the index operator to get a specific device:

from Bass4Py.bass import BASS

bass = BASS()
device = bass.input_devices[0]

This attribute will work as an iterator too, so that you can loop over all the available devices on your system.

from Bass4Py.bass import BASS

bass = BASS()

for device in bass.input_devices:
    print(device.name)

Creating a list of all the devices works just the same, you can use the list for that purpose.
