OutputDevice¶

class Bass4Py.bass.OutputDevice¶

Control several settings related to your specific output device and create new audio streams.

An output device allows to configure several device-related settings, read device-related information like name and create streams or sample, e.g. with the create_stream_from_file() method.

Before a device can actively be used, it must be initialized via the init() method. A few things can be done without doing so though, mostly accessing some attributes like the name attribute.
