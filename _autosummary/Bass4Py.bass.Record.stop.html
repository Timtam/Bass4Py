Record.stop¶

Record.stop(self)¶

Stops this channel.

Returns

success

Return type

bool

Stopping a user stream (created with Bass4Py.bass.OutputDevice.create_stream_from_parameters()) will clear its buffer contents, and stopping a Bass4Py.bass.Sample channel or Bass4Py.bass.Record will result in it being freed. Use pause() instead if you wish to be able to resume them (with Bass4Py.bass.Channel.play()). When used with a decoding channel, this function will end the channel at its current position, so that it is not possible to decode any more data from it. Any Bass4Py.bass.syncs.End syncs that have been set on the channel will not be triggered by this; they are only triggered when reaching the natural end. Bass4Py.bass.Channel.set_position() can be used to reset the channel and start decoding again.
