BASS.net_playlist¶

Process URLs in PLS and M3U playlists?

When to process URLs in PLS and M3U playlists… 0 = never, 1 = in Bass4Py.bass.OutputDevice.create_stream_from_url() only, 2 = in Bass4Py.bass.OutputDevice.create_stream_from_file() and Bass4Py.bass.OutputDevice.create_stream_from_file_obj() too.

When enabled, BASS will process PLS and M3U playlists, trying each URL until it finds one that it can play. Bass4Py.bass.Stream.name can be used to find out the URL that was successfully opened. Nested playlists are suported, that is a playlist can contain the URL of another playlist. The Bass4Py.bass.BASS.net_playlist_depth option limits the nested playlist depth. By default, playlist processing is disabled.
