Chorus.set¶

Chorus.set(self, Channel chan)¶

Apply this effect to a channel.

Parameters

chan (Bass4Py.bass.Channel) – one of the various channel implementations.

Raises

  • Bass4Py.exceptions.BassNoFXError – The specified DX8 effect is unavailable.

  • Bass4Py.exceptions.BassFormatError – The channel’s format is not supported by the effect.

  • Bass4Py.exceptions.BassUnknownError – Some other mystery problem!

Multiple effects may be used per channel. Use remove() to remove an effect. Use the effect-specific attributes to set an effect’s parameters. An effect’s priority value can be changed via the priority attribute. Effects can be applied to Bass4Py.bass.Music and Bass4Py.bass.Stream, but not Bass4Py.bass.Sample. If you want to apply an effect to a sample, you could use a stream instead.

Platform-specific

DX8 effects are a Windows feature requiring DirectX 8, or DirectX 9 for floating-point support. On other platforms, they are emulated by BASS, except for the following which are currently unsupported: Bass4Py.bass.effects.dx8.Compressor, Bass4Py.bass.effects.dx8.Gargle, and Bass4Py.bass.effects.dx8.I3DL2Reverb.
