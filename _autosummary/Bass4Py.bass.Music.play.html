Music.play¶

Music.play(self, bool restart=False)¶

Starts (or resumes) playback.

Parameters

restart (bool) – Restart playback from the beginning? If this is a user stream (created with Bass4Py.bass.OutputDevice.create_stream_from_parameters() or Bass4Py.bass.Stream.from_parameters()), its current buffer contents are cleared. If this is a Bass4Py.bass.Music, its BPM/etc are reset to their initial values.

Returns

success

Return type

bool

Raises

  • Bass4Py.exceptions.BassStartError – The output is paused/stopped, use Bass4Py.bass.OutputDevice.start() to start it.

  • Bass4Py.exceptions.BassDecodeError – The channel is not playable; it is a “decoding channel”.

When streaming in blocks (Bass4Py.constants.STREAM.BLOCK), the restart parameter is ignored as it is not possible to go back to the start. The restart parameter is also of no consequence with recording channels. If other channels have been linked to the specified channel via link(), this function will attempt to simultaneously start playing them too but if any fail, it will be silently. The exception raised only reflects what happened with the specified channel. Bass4Py.bass.Channel.active can be used to confirm the status of linked channels.
