OutputDevice.free¶

OutputDevice.free(self)¶

Frees all resources used by the output device, including all its samples, streams and MOD musics.

Returns

success

Return type

bool

Raises

Bass4Py.exceptions.BassInitError – init() has not been successfully called.

This function should be called for all initialized devices before the program closes. It is not necessary to individually free the samples/ streams/ musics as these are all automatically freed by this method.
